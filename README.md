# AIKYAM Receipt Generator

Automatically generate personalized PDF receipts from an Excel sheet and email them to donors.  
This tool is designed for AIKYAM to streamline invoice generation and distribution for community events.

---
## Features
- Generates professional PDF receipts using a letterhead template  
- Sends receipts automatically to donor emails  
- Formats invoice date, phone number, and amount  
- Saves all generated PDFs neatly in a `/receipts` folder  
- Uses Gmail App Password for secure authentication
